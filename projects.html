Personal Projects

In this section I talk about some of the personal projects that I am proud of

HashTag Creator – IOS App

I have created a IOS app that is based on ML Models from Apple. In this app the user may upload or take a photo and then the app will guess what is in this picture, and then it will give top 10 hashtags for this prediction. Please take a look at my GitHub repository if you would like to know more.

Poker Help – Alexa Skill

Wrote an Alexa skill by writing a Lambda function in AWS, which allows the user to ask Alexa whether they should keep or fold their hand in Poker. This project was done during the Hackathon.

Name Generator

I have created an application that can return novel male and female names based on the dataset of 2,000 unique names. The way my application can generate new names is based on Markov Model. If you would like to know more, please check out my GitHub repository

Eight Puzzle

I have created an Goal based Agent, that will solve the Eight Puzzle given the randomized array of numbers. To be more specific this application will take an array of numbers such as [1, 2, 4, 5, 3, 8, 7, 6, 0] and will change the order of numbers until it reaches the goal of [1, 2, 3, 8, 0, 4, 7, 6, 5].

Calculator IOS App

I have created a very simple Calculator App for IOS. This was my first project in IOS Development, so it very simple. However, it is still pretty cool, as it can perform simple calculations.

Remote – IOS App

I have created a Remote app, which is just a mockup to what a Remote app can look like, since it is not connected to TV anyhow. However, this app still show several important concepts of IOS Development, such as multi-view application where data is shared between each view. In addition, this app also shows the concepts of autolayout.

Blockchain Project

This was my Senior Capstone Porject, where I was part of 7 Man Team, in which I was responsible for Collaboration Software, Planning, Testing and Implementation roles. This project was a Blockchain based on Health Records. By the end of this project, we were able to show a command line interface blockchain, which was ran on multiple servers. On top of that, we have built a frontend, which was a React JS App.

Scala Mini Compiler

Wrote a mini lisp compiler in scala. This project included hands on experience on several topics such as, AST, parsing, and assembly language, as well as minimal understanding of lisp.
